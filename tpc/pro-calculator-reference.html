Rene Herse Cycles Tire Pressure Calculator  ·  How each input affects the pressure recommendation

Overall Formula

Each wheel's pressure is calculated through a pipeline:

Base pressure (from equations) × scaling factor (wheel load / total weight + complement) × terrain multiplier + casing adjustment × rim width multiplier

The base pressure equation is: P (psi) = slope × total weight (kg) + intercept, where slope and intercept depend on tire width and feel (firm vs. soft).

For a standard road bike with neutral riding position and no panniers, the Pro calculator produces the same result as the Simple calculator.

Tire Width (Front / Rear)

Selects which set of equation coefficients to use. The equations are anchored at specific widths (25, 28, 35, 38, 44, 48, 55 mm) and linearly interpolated for values in between.

  • A wider tire → lower slope and different intercept → lower base pressure
  • Also divides the casing adjustment value, so a given casing type has a smaller absolute effect on wider tires

Rider Weight / Bike Weight

  • Together they form the total system weight, which is the input to the pressure equation — the same weight the Simple calculator uses
  • Also distributed front/rear via Bike Style and Riding Position to compute per-wheel loads
  • Higher total weight → higher base pressure

Front Panniers / Rear Panniers

  • Added directly to that wheel's load only — not distributed between wheels
  • Also included in the total system weight, so they raise the base pressure and increase that wheel's scaling factor
frontLoad = (rider + bike + bikepacking) × (frontRatio + positionAdj) + frontPanniers rearLoad = (rider + bike + bikepacking) × (rearRatio + positionAdj) + rearPanniers

Bikepacking Weight

  • Distributed front/rear according to the Bike Style and Riding Position ratios — the same way rider and bike weight are split
  • Also included in the total system weight

The combined scaling factor for each wheel is:

frontScale = frontLoad / totalWeight + 0.60 (road rear complement) rearScale = rearLoad / totalWeight + 0.40 (road front complement)

A road bike with neutral position gives a scale of 1.00 for both wheels, identical to the Simple calculator.

Feel (Soft / Firm)

  • Selects which equation coefficients to use — firm equations generally produce higher pressure than soft at the same weight and width
  • Overridden to Soft whenever terrain is any gravel option

Casing (Front / Rear)

An additive pressure adjustment applied after scaling and terrain, before the rim width multiplier.

Rim Width

Reduces pressure for rims wider than 19 mm. Capped at 32 mm (wider rims produce no further reduction).

rimMultiplier = 1 − (min(rimWidth, 32) − 19) × 0.002 [only when rimWidth > 19 mm, else 1.0]

Automatic Limits (not user-controlled)

Front minimum pressure

After all other calculations, if the front pressure falls below the soft-feel pressure for 50% of total system weight, the front result is set to N/A with a braking performance warning.

frontMinimum = softEquation(totalWeight × 0.5, frontTireWidth)
Note: When a result shows N/A, it means the calculated pressure for that wheel exceeds a safety limit (hookless or tubeless cap) or falls below the front minimum. The calculator does not clamp to the nearest legal value — N/A means the recommended pressure cannot be safely achieved with the current setup.
